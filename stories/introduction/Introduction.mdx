# Ring MUI Components

#### Shared internal React components and foundations built on top of Material UI.

#### This Storybook provides reusable building blocks, live examples, migration notes, and contribution standards for Ring applications.

**Requirements:** Node 18+ · React 18+ · Material UI 7+

> **Note:** Ring Publishing MUI Components are not intended to function as a general-purpose UI toolkit — they are only useful in the context of integrating with the Ring Publishing system.

## About the library

MUI Components is a shared internal library built on top of MUI.

It extends the base MUI ecosystem with internal components, theme integration, and shared documentation standards.

Use this Storybook to:

- explore available shared components,
- review usage examples and API documentation,
- check migration notes for breaking changes,
- follow contribution standards for new components and stories.

For base functionality that belongs to MUI itself, use the [official MUI documentation](https://mui.com/material-ui/all-components/) together with this Storybook.

---

## Environment

MUI Components supports **Node 18+** and **React 18+**.

Some components depend on selected MUI and MUI X packages.
Install only the peer dependencies required by the components you use.

Detailed installation, theme setup, and licensing instructions are available on the [Setup](?path=/docs/introduction-setup--docs) page.
