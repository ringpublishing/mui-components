# Alert

Alert is a structured wrapper over MUI Alert. It uses explicit `title` and `description`, a controlled action button config, and supports both `outline` and `inline` frame variants.

## Key Features

- `title` and `description` are first-class content props
- action is a controlled button config, not an arbitrary node
- the close X is rendered from `onClose`
- supports `actionsPlacement="right" | "bottom"`
- supports `layoutVariant="outline" | "inline"`
