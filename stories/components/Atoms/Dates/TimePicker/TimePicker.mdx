# TimePicker

A time selection component that wraps MUI's [TimePicker](https://mui.com/x/react-date-pickers/time-picker/) with Ring Design System styling. Provides a clock popup for selecting hours and minutes with built-in support for clearable input.

- Clearable input with clock icon
- Configurable label
- Extends all MUI TimePicker props
- Supports `sx` and `className` for custom styling

## Prerequisites

TimePicker requires a `LocalizationProvider` wrapper with a date adapter. Set this up once at the root of your application:

```tsx
import { LocalizationProvider } from '@mui/x-date-pickers-pro';
import { AdapterDayjs } from '@mui/x-date-pickers-pro/AdapterDayjs';
import 'dayjs/locale/en';

function App() {
    return (
        <LocalizationProvider dateAdapter={AdapterDayjs} adapterLocale="en">
            {/* Your app content */}
        </LocalizationProvider>
    );
}
```

## Outlined

Use the `outlined` variant via `slotProps.textField` for a bordered input style. The floating label appears inside the border and moves to the top edge when focused or when a value is present.

```tsx
<TimePicker
    label="Time"
    slotProps={{
        textField: {
            variant: 'outlined',
        },
    }}
/>
```
