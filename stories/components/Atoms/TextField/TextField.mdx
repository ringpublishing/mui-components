# TextField

TextField extends [MUI Text Field](https://mui.com/material-ui/react-text-field) component and supports all its properties. It adds support for contextual **actions** rendered as icon buttons in the field's end adornment.

**Key Features:**

- Single action rendered as a standalone icon button with tooltip
- Multiple actions collapsed into a `MoreVert` menu via `ActionBox`
- Full MUI TextField API support (variants, sizes, states, etc.)
- Controlled and uncontrolled usage patterns
