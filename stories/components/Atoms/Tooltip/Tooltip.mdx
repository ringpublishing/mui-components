# Tooltip

An enhanced tooltip component built on MUI Tooltip that supports structured content with a title, optional subtitle, and optional hint.

**Key Features:**

- Structured content with title, subtitle, and hint sections
- Full MUI Tooltip API compatibility (placement, arrow, delay, etc.)
- Keyboard hint display for discoverability
