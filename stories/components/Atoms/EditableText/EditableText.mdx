# EditableText

An inline editable text component that renders as plain text with an edit button, and switches to a text field when the user activates edit mode.

**Key Features:**

- Toggle between display and edit mode with a single click
- Submit changes on click away or Enter key
- Cancel changes with the Escape key
- Automatic revert on failed submission
- Optional label for the input field in edit mode
- Customizable internal slots via `slotProps` (Typography and TextField)

**Keyboard shortcuts (in edit mode):**

- `Enter` — submit changes
- `Escape` — cancel and revert to previous value
