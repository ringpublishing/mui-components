# SearchBar

SearchBar is a comprehensive search component that combines a search input field with customizable action buttons and controls. It supports both controlled and uncontrolled modes, debounced search functionality, and a clearable input option.

- Flexible search input with debouncing support
- Customizable action buttons and controls
- Clear button for quick input reset
- Supports both controlled and uncontrolled modes
- Built on Material-UI with flexible styling
