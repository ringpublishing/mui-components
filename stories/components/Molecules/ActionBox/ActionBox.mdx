# ActionBox

ActionBox displays a list of actions in a compact dropdown menu anchored to a trigger element.
It supports icons, separators, disabled state handling, and tooltips for disabled reasons.

**Main use cases:**

- Context menus with additional options
- Multiple actions in a compact form
- Conditional actions with disabled state and reasons
- Organizing functions with separators

ActionBox extends [Popper](https://mui.com/material-ui/react-popper/), so all Popper props are supported.
