# Placeholder

`Placeholder` displays an empty/error-like state with an icon and contextual text to help users understand what happened and what they can do next.

- Supports predefined variants for common scenarios (error, not found, empty)
- Supports custom labels for localized or domain-specific messaging
- Supports optional action buttons and a custom image/icon
- Uses theme locale for localized defaults
