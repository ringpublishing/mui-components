# SearchBox

SearchBox is a lightweight search component that provides a simple search input field with customizable styling and behavior. It supports both controlled and uncontrolled modes, debounced search functionality, and a clearable input option. The component is built on top of Material-UI and provides flexible styling options through className and sx props.

- Debounced search input with customizable delay
- Clearable input with optional clear button
- Supports both controlled and uncontrolled modes
- Customizable labels and styling
