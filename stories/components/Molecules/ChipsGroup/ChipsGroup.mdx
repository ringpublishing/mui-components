# ChipsGroup

The ChipsGroup component displays a collection of Chip components with intelligent automatic overflow handling. When chips don't fit in the available space, the component shows a "+N" indicator for hidden chips.

## Features

- **Automatic overflow handling**: Dynamically calculates visible chips and shows "+N" button for hidden ones
- **Flexible display modes**:
    - Default mode shows hidden chips in a dropdown menu
    - Expandable mode displays hidden chips inline when "+N" is clicked
    - Collapsable mode allows returning to the compact view after expansion
- **Delete all functionality**: Optional button to clear all chips at once
- **Customizable labels**: Override default text for "Clear" and "Show less" buttons
- **Responsive**: Adapts to container width changes
- **Individual chip actions**: Each chip can have its own click, delete, and styling

## Chips Expandable

Enable inline expansion of hidden chips by setting `expandable={true}`. When the user clicks the "+N" button, all hidden chips are displayed inline rather than in a dropdown menu.

### Key Features:

- Click "+N" to expand and show all chips inline
- Useful when you want all chips visible at once
- Better for scenarios where users need to see all options simultaneously

<Canvas of={Stories.ChipsExpandable} />

## Chips Expandable and Collapsable

Combine `expandable` and `collapsable` props to allow users to toggle between compact and expanded views. After expanding, a "Show less" button appears to collapse the chips back to the original state.

### Key Features:

- Expand chips inline with "+N" button
- Collapse back with "Show less" button
- Gives users control over the view density

## Delete All

Add a delete all button by providing the `onDeleteAll` callback. This displays a "Clear" button at the end of the chips group, allowing users to remove all chips with a single click.

### Usage:

```typescript
<ChipsGroup
  chips={chips}
  onDeleteAll={() => handleClearAll()}
/>
```

<Canvas of={Stories.DeleteAll} />

## Custom Labels

Customize the text for action buttons using the `customLabels` prop. This is useful for internationalization or matching your application's terminology.

### Available Labels:

- `deleteAll`: Text for the delete all button (default: "Clear")
- `showLess`: Text for the collapse button (default: "Show less")

### Usage:

```typescript
<ChipsGroup
  chips={chips}
  expandable
  collapsable
  onDeleteAll={handleClearAll}
  customLabels={{
    deleteAll: 'Delete all chips',
    showLess: 'Hide'
  }}
/>
```
