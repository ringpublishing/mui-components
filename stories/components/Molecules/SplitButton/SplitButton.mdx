# SplitButton

SplitButton is a specialized button component that combines a primary action with a dropdown menu of additional actions. It extends the MUI ButtonGroup component with enhanced functionality for managing multiple related actions.

- Primary action button with optional disabled state
- Dropdown menu for additional actions with icons and tooltips
- Tooltip support for disabled action reasons
- Customizable sizing and styling via ButtonGroup API
- Support for single or multiple additional actions
