# FeatureTooltip

A feature announcement tooltip that automatically manages its display state using `localStorage`. It extends MUI's [Tooltip](https://mui.com/material-ui/react-tooltip/) with a rich card layout, supporting media content, configurable action buttons, and frequency-capping logic.

- Displays automatically on mount based on `capping` count and `endDate` expiration
- Persists display history in `localStorage` to control re-display frequency
- Supports text messages, embedded videos (`videoEmbed`), and images (`image`)
- Configurable action buttons — up to two: primary and optional secondary
- Close button always available for the user

## Default

Shows two tooltip configurations side by side: a **standard tooltip** (displayed only once per user) and a **capping tooltip** (displayed up to 3 times). Both use `localStorage` to track display history.

> **Note:** If you don't see the tooltip, click **Clear tooltip data** to reset the `localStorage` state and reload the page.
