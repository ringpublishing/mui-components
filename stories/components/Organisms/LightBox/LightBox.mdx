# LightBox

A full-featured image viewer dialog with carousel navigation, zoom controls, and fullscreen support. Displays images in a separate dialog/gallery with smooth transitions and touch gesture support.

- Navigate through multiple images with arrow keys or carousel
- Zoom in/out with mouse wheel or pinch gestures
- Fullscreen mode support
- Image download capability with custom handler option
- Optional detail panel with metadata, actions, and editable title
- Infinite scroll for loading additional images on demand
- Keyboard navigation (arrow keys, Escape to close)

## With Detail

Shows a detail panel alongside the image with metadata sections, action buttons, and an editable title. The detail panel supports description items with various field types (text, chips, dates) and bottom action buttons.

### Key Features:

- Editable title field
- Description items organized in sections
- Action buttons with icons and URLs or click handlers
- Collapsable chip fields for tags
