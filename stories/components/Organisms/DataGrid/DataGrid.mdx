# DataGrid

An enhanced data grid component built on top of MUI X DataGrid.
**Features:**

- Custom toolbar with refresh, auto-refresh toggle, sorting, and additional components
- Bulk actions mode with select all/deselect all and configurable max selection
- Built-in custom cell renderers: combo cell, link cell, action cell, and spacer rows
- Auto-refresh with configurable rate, user interaction debounce, and initial state
- Filter chips group for active filter display
- Checkbox selection with optional disabled select-all header
- Error and no-results placeholder states
