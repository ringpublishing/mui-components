# TreeView

The TreeView component displays hierarchical data in an expandable/collapsible tree structure. It supports two variants: **default** (with columns, checkboxes, row actions, and drag-and-drop) and **compact** (a minimal layout with custom elements).

**Key Features:**

- Two layout variants: `default` (tabular with columns) and `compact` (minimal)
- Expandable/collapsible nested items
- Search and filter with word-based matching and highlighting
- Checkbox selection with disable capability (default variant)
- Drag and drop reordering (default variant)
- Drop-in nesting to create new hierarchy levels (default variant)
- Dynamic/lazy loading of child items with error handling and retry
- Row action buttons (default variant)
- Controlled and uncontrolled selection modes
